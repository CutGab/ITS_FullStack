###########
#         #
#  ARRAY  #
#         #
###########

Gli array consentono di associare **più valori a un unico nome di variabile**, permettendo di gestire insiemi di dati correlati come un’unica entità. 
In genere, i valori contenuti in un array condividono una stessa natura o uno stesso significato, ad esempio un elenco di giorni, mesi, nomi o numeri.
L’utilizzo degli array evita la dichiarazione di molte variabili separate e rende molto più semplice eseguire operazioni ripetitive o cicliche sugli elementi.

```
var giorniDellaSettimana = [
    "lunedì",
    "martedì",
    "mercoledì",
    "giovedì",
    "venerdì",
    "sabato",
    "domenica"
];
```

I valori di un array sono racchiusi tra **parentesi quadre** e separati da virgole. Gli array in JavaScript sono **dinamici**, 
cioè possono crescere o ridursi automaticamente senza dover specificare una dimensione iniziale.

Ogni valore all’interno di un array è chiamato **elemento** e possiede una posizione numerica detta **indice**. Gli indici partono sempre da `0`.

```
console.log(mesi[2]);
var meseScelto = mesi[2];
```

Per accedere a un elemento è sufficiente indicare il nome dell’array seguito dall’indice tra parentesi quadre.

```
var primoGiorno = giorniDellaSettimana[0];
```

Gli array possono contenere elementi di **qualsiasi tipo**, anche misti:

```
var mioArray = [123, "stringa", true, null];
```

Per conoscere il numero di elementi presenti in un array si utilizza la proprietà `length`:

```
console.log(giorniDellaSettimana.length);
```

##########################
#                        #
#  OPERATORI ARITMETICI  #
#                        #
##########################

Gli operatori aritmetici permettono di eseguire operazioni matematiche sui numeri. Sono operatori binari che lavorano su due operandi.

```
+  addizione
-  sottrazione
*  moltiplicazione
/  divisione
%  modulo (resto)
```

Esistono anche operatori di assegnamento combinati, che rappresentano una forma abbreviata:

```
x += y   // x = x + y
x -= y   // x = x - y
x *= y   // x = x * y
x /= y   // x = x / y
x %= y   // x = x % y
```

JavaScript supporta inoltre operatori **unari**:

```
+   cast a numero
-   negazione
++  incremento
--  decremento
```

La posizione degli operatori di incremento e decremento è fondamentale:

```
var x = 3;
var y = ++x; // x = 4, y = 4

var x = 3;
var y = x++; // x = 4, y = 3
```

Gli operatori **relazionali** consentono il confronto tra valori:

```
<   minore
<=  minore o uguale
>   maggiore
>=  maggiore o uguale
==  uguale (valore)
!=  diverso (valore)
=== strettamente uguale (tipo e valore)
!== strettamente diverso (tipo e valore)
```

Il risultato di un confronto è sempre un valore booleano.

```
4 > 2        // true
5 != 4 + 1  // false
```

Gli operatori **logici** permettono di combinare espressioni booleane:

```
&&  AND
||  OR
!   NOT

5 > 2 && 3 != 4   // true
true || false    // true
!true            // false
```

#######################################
#                                     #
#  CONVERSIONE FRA TIPI DI VARIABILI  #
#                                     #
#######################################

JavaScript consente alle variabili di cambiare tipo dinamicamente. In molte situazioni il motore JavaScript effettua **conversioni implicite** per rendere valide le operazioni.

```
var x = "12";
var y = "13";
var z = x * y; // 156
```

In questo caso le stringhe vengono convertite automaticamente in numeri. Tuttavia, in altre situazioni il comportamento può essere ambiguo:

```
var x = "12";
var y = x + 13; // "1213"
```

Conoscere le regole di conversione implicita è fondamentale per evitare bug difficili da individuare.

############
#          #
#  PROMPT  #
#          #
############

La funzione globale `prompt()` permette di mostrare una finestra di dialogo all’utente, chiedendo l’inserimento di un valore.

Se l’utente conferma, il valore restituito è una **stringa**; se annulla, il valore restituito è `null`.

```
var nome = prompt("inserisci il tuo nome");
console.log(nome);
```
