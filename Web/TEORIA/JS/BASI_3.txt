#                          #
# ISTRUZIONI CONDIZIONALI  #
#                          #
############################

Le istruzioni condizionali sono costrutti fondamentali del linguaggio JavaScript che permettono di **controllare il flusso di esecuzione** di un programma. 
Grazie a queste istruzioni, uno script può eseguire blocchi di codice differenti in base al risultato di una condizione, 
simulando una vera e propria presa di decisione.

In JavaScript sono disponibili due principali istruzioni condizionali:

```
- if
- switch
```

L’istruzione if può presentarsi in tre forme principali:

```
- if semplice;
- if con alternativa (if..else);
- if a cascata (if..else if…else).
```

La forma più semplice di if consente di eseguire un blocco di codice **solo se una condizione risulta vera**:

```
if (condizione) {
    // istruzioni
}
```

In questo caso *condizione* è un’espressione booleana (true o false), mentre *istruzioni* 
rappresenta il codice che verrà eseguito esclusivamente quando la condizione è verificata.

La seconda forma è l’istruzione if...else. In questo caso vengono definiti due percorsi alternativi di esecuzione:

```
if (condizione) {
    // istruzioni1
} else {
    // istruzioni2
}
```

Se la condizione è vera, JavaScript esegue il blocco *istruzioni1*; in caso contrario viene eseguito il blocco *istruzioni2*. Questa struttura è utile quando è necessario gestire sia il caso positivo sia quello negativo di una condizione.

La terza forma è l’if a cascata (if..else if...else), che consente di gestire **più condizioni alternative**:

```
if (condizione1) {
    istruzioni1
} else if (condizione2) {
    istruzioni2
} else {
    istruzioni3
}
```

In questo caso JavaScript valuta le condizioni dall’alto verso il basso. Il primo blocco la cui condizione risulta vera viene eseguito, mentre gli altri vengono ignorati. Il blocco else finale è opzionale e viene eseguito solo se nessuna delle condizioni precedenti è soddisfatta.

################
#              #
# SWITCH CASE  #
#              #
################

L’istruzione switch rappresenta un’alternativa all’uso di lunghe catene di if...else if, ed è particolarmente indicata quando si deve confrontare **il valore di una singola espressione** con più casi possibili.

Lo switch migliora la leggibilità del codice quando le alternative sono numerose ed è spesso più efficiente, poiché il motore JavaScript può individuare direttamente il caso corretto senza valutare tutte le condizioni in sequenza.

```
switch (espressione) {
    
    case espressione1:
        istruzioni1;
        break;
    
    case espressione2:
        istruzioni2;
        break;
    
    default:
        istruzioni3;
        break;
}
```

L’espressione viene valutata una sola volta e confrontata con i vari case. Quando viene trovato un caso corrispondente, il codice associato viene eseguito fino a incontrare l’istruzione break, che interrompe lo switch.

Il blocco default è facoltativo e viene eseguito se nessun case corrisponde al valore dell’espressione.

È possibile omettere volontariamente il break per eseguire lo stesso blocco di codice per più valori consecutivi, comportamento noto come *fall-through*.

#########################
#                       #
# WHILE, DO-WHILE, FOR  #
#                       #
#########################

Le istruzioni di iterazione permettono di eseguire ripetutamente un blocco di codice finché una determinata condizione risulta vera. JavaScript mette a disposizione tre principali strutture cicliche: while, do-while e for.

Il ciclo while esegue il blocco di codice **finché la condizione è vera**:

```
while (condizione) {
    // istruzioni
}
```

La condizione viene valutata prima di ogni iterazione. È fondamentale che il codice all’interno del ciclo modifichi la condizione, altrimenti si rischia di creare un ciclo infinito.

Una variante del while è il do-while:

```
do {
    // istruzioni
} while (condizione);
```

In questo caso la condizione viene valutata **dopo** l’esecuzione del blocco di codice. Questo garantisce che le istruzioni vengano eseguite almeno una volta, anche se la condizione inizialmente è falsa.

Il ciclo for è generalmente utilizzato quando il numero di iterazioni è noto o facilmente determinabile:

```
for (inizializzazione; condizione; modifica) {
    // istruzioni
}
```

* inizializzazione: viene eseguita una sola volta prima dell’inizio del ciclo;
* condizione: viene valutata prima di ogni iterazione;
* modifica: viene eseguita al termine di ogni iterazione.

Per lavorare più comodamente con gli array, JavaScript fornisce due varianti del ciclo for: for-in e for-of.

Il for-in itera sugli indici dell’array:

```
var quantita = [12, 24, 42, 11, 29];
var totale = 0;

for (var indice in quantita) {
    totale = totale + quantita[indice];
}
```

Il for-of, invece, itera direttamente sui valori contenuti nell’array, rendendo il codice più leggibile:

```
var quantita = [12, 24, 42, 11, 29];
var totale = 0;

for (var valore of quantita) {
    totale = totale + valore;
}
```
