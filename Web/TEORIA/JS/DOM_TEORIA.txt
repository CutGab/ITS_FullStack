#########
#       #
#  DOM  #
#       #
#########

Il DOM (Document Object Model) fornisce una rappresentazione del documento come una struttura gerarchica ad albero di oggetti.

Questo albero è progettato per essere facilmente accessibile tramite JavaScript non solo in lettura, ma anche per modificarne dinamicamente la struttura, 
il contenuto e lo stile.

In altre parole, il DOM è un insieme di interfacce (API), funzioni, metodi e proprietà che i programmi possono utilizzare per manipolare il documento.

Il DOM non fa parte di JavaScript: esso è un modello indipendente, e JavaScript è uno dei linguaggi che possono interagire con esso.

#####################
#                   #
#  CARATTERISTICHE  #
#                   #
#####################

1. Document: rappresenta l’intero documento HTML ed è l’oggetto radice del DOM.
2. Elements: permette di selezionare e modificare elementi specifici.
3. Attributes: consente di leggere o modificare attributi degli elementi.
4. Events: gestione degli eventi legati agli elementi.
5. Style: modifica degli stili CSS applicati agli elementi.

Metodi principali dell’oggetto `document`:

    - getElementById(id)
    - getElementsByTagName(tag)
    - getElementsByClassName(classe)
    - getElementsByName(nome)

Alcune proprietà utili dell’oggetto `document`:

    - document.URL
    - document.head
    - document.body
    - document.links

Esempio di selezione tramite `id`:

    <p id="mioParagrafo">Testo</p>

    var elemento = document.getElementById("mioParagrafo");

Se l’elemento non esiste, il metodo ritorna `null`.

Altri metodi di selezione:

    document.getElementsByTagName("tag")      // Collezione di elementi per tag
    document.getElementsByClassName("classe") // Collezione di elementi per classe
    document.getElementsByName("nome")        // Collezione di elementi per nome

############
#          #
#  EVENTI  #
#          #
############

Gli eventi DOM consentono di eseguire codice JavaScript in risposta a interazioni con la pagina, come click, passaggio del mouse o invio di form.

Esempio semplice con attributo HTML:

    <button onclick="alert('ciao');">Salutami</button>

Esempio con funzione separata:

    <script>
        function saluta() {
            alert('ciao');
        }
    </script>

    <button onclick="saluta();">Salutami</button>

Gestione tramite `getElementById` nello script:

    <button id="miobottone">Salutami</button>

    <script>
        document.getElementById("miobottone").onclick = function() {
            alert('ciao');
        }
    </script>

Gestione più pulita separando variabili e funzioni:

    <button id="miobottone">Salutami</button>

    <script>
        var mio = document.getElementById("miobottone");

        mio.onclick = function() {
            alert('ciao');
        }
    </script>

Alternativa moderna con `addEventListener`:

    <h2 id="paperino">dom</h2>
    <button id="miobottone">Salutami</button>

    <script>
        var mio = document.getElementById("miobottone");

        mio.addEventListener("mouseover", primafunzione);
        mio.addEventListener("click", secondafunzione);
        mio.addEventListener("mouseout", terzafunzione);

        function primafunzione() {
            document.getElementById("paperino").innerHTML = 'ho generato un mouseover';
        }

        function secondafunzione() {
            document.getElementById("paperino").innerHTML = 'ho generato un click';
        }

        function terzafunzione() {
            document.getElementById("paperino").innerHTML = 'ho generato un mouseout';
        }
    </script>
